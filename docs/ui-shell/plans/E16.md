# E16 тАФ Recipes + Snippets тАФ Implementation Plan

> Epic E16. Goal: build the **Recipes** subsystem (save/restore layouts + command
> replay + trust + shell-handoff pause) and surface the existing **snippet** engine
> (settings editor + alias + reserved template vars + palette).
>
> **Wire posture: `touchesWire = false`** тАФ everything is client-side. No new wire
> message, no metadata verb. Golden corpus MUST stay zero-diff (33 emitted
> byte-identical + 13 frozen keys intact). `touchesIOS = true`. `touchesFloatCodec
> = false` (no codec/controller/SIMD touched; one numeric `size`/`weight` clamp uses
> ordered min/max, never FMA).
>
> Primary evidence read: `BACKLOG.md` (E16), `plans/E16-carryovers.md` (binding
> acceptance gates), `spec/customization__custom-commands.md` (authoritative `.slopdeskrecipe`
> TOML example, replay-mode table, trust rules, snippet placeholder table + the two
> screenshots), `spec/user-interface__window-tab-split.md`, USER-STORIES
> ES-E16-1тАж4, current-state `workspace-domain.md` / `settings-config.md` /
